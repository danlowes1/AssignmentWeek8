        ## Description

        Backend solution with CRUD operations

        ## Installation

        For dev usage install the following packages:
        npm i
        jsonwebtoken
        sequelize
        express
        nodemon
        concurrently 
        mysql2
        bcrypt

        To run SQL script:
        mysql -u root -p
        Run script: source db/schema.sql;
        npm run start rebuild --rebuildnpm
